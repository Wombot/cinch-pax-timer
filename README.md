# Cinch::Pax::Timer

Cinch Plugin to track th time till PAX.

## Installation

Add this line to your application's Gemfile:

    gem 'cinch-pax-timer'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cinch-pax-timer
